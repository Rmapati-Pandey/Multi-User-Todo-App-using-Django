Django-ToDo-App

Document

This is a To-Do app build using django framework of python. Here user can create his todo list by adding items, crossed off the completed items, delete the completed items and can delete all the items.

Technologies Used :

  • Python
  • Django
  • Bootstrap
  • db.sqlite3

Additional Python Modules Required:

1.Django Running the project: 1. Migrate : > cd todo_app > python manage.py migrate 2. Make Admin User : > python manage.py createsuperuser 3. runserver : > cd todo_app > python manage.py runserver
